# Levels

Levels are data-driven tile maps loaded by the shared gameplay scene.

- `tutorial.ts`: Level 1, First Jar
- `levelTwo.ts`: Level 2, Over and Under
- `levelThree.ts`: Level 3, The Long Way Round (six-room dungeon)

Tile vocabulary:

- `#` solid ground
- `.` empty space
- `S` player spawn
- `F` mineral pickup
- `E` exit
- `^` hazard
